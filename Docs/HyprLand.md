
Install X11
```
# open terminal as root user
pacman -S xorg
pacman -S xorg-xinit xorg-twm xorg-xclock xterm

# run x 
startx

# exit
exit
```


Install from pacman

```
sudo pacman -S hyprland kitty gtk3

```

config hyprland
```
vim /.config/hypr/hyprland.conf
```
